# Download-doc
download documents from academia.edu, issuu, slideshare with Python-selenium 
instructions:

## Instructions :
### - sudo apt install python3-pip
### - pip3 install bs4
### - pip3 install selenium
### - pip3 install img2pdf
### - python3 main.py
